# cli-trivia-game
This is a Command-line interface trivia game built using javascript and Node.js.Players can answer questions,get feedback and see their final score along with the time taken. 

---
## How does it work
-You run the game in the terminal with node quiz.js.
-The game print a welcoming message.
-It start tracking time using Date.now().
-The game uses an array of question objects.
-The game loops through this array and displays each question with numbered options.
-The Node.js readline module waits for the user to enter a number for their chosen answer.
-It then compared the selected option to the correct one.
-After feedback, the game automatically goes to the next question.
-This repeats until all questions in the array have been answered.
-Once all questions are answered, the game calculates total time taken.
-Displays final score and time spent.
-The readline interface is closed.
